## Сопоставление данных о товарах на Ozon с остатками и ценами часов Casio

Скрипт seller.py получает все артикулы товаров, загруженные в Ozon. Скачивает остатки часов с сайта поставщика Сasio. Далее происходит сопоставление остатков товаров с артикулами Ozon, определяется количество товаров в наличии. Обновленные остатки отправляются в Ozon (партиями по 100 товаров). Формируются цены на основе данных с сайта поставщика, после чего происходит обновление цен в Ozon (партиями по 900 товаров).
